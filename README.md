# expense-laravel

## Description
Simple API REST CRUD using Laravel 9.

The project contains:
<ul>
    <li>Form Request Validation</li>
    <li>Repository</li>
    <li>API Resource Routes</li>
    <li>Unit and Feature tests</li>
    <li>Event and listener</li>
    <li>Queue</li>
    <li>Email notification</li>
    <li>Docker</li>
</ul>
